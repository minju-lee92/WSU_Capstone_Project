Where the Data Came From

The data comes from SEL(Schweizer Engineering Laboratories), It contains three years of raw data for 100 customers of vareid sizes. The information we are working on is fictitious, due to confidentiality of the data.

Special Properties of the Data

Our dataset contains SEL customer orders for three years and is displayed over five tables. -The first table is Customer Info which has data for overall customer ID which represent the customers overall place in which that customer became a customer with SEL, the customer ID numbers which is the identity number for that customer, and customer size. The customer sizes have several levels ranging from -1 to 27. -Table two is Table A which contains data on customer location ID numbers, date order was placed, and order type. Order type is a numeric value that is either 0 or 1 which indicated order types A or B respectively. -The third table is Table B which contains order status information. This has customer ID numbers, order dates, the total number of items ordered, and then the number of items that were received by the customers early, on time, and late. -Table four is Table D which is similar to Table A, the only difference is that this table gives customer ID numbers instead of customer location ID numbers. Type values are the same as Table A as well. -The final table is Table E which contains company ID numbers which is the same as customer ID numbers from Table D, dates orders were placed and completed, and order type. Order type is indicated by 0 or 1 which corresponds to types A or B respectively. -Many of the tables share common column values which helps to link them to give different pieces of information about each customer order.
